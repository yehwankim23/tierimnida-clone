# [TIERIMNIDA](https://github.com/mazassumnida/tierimnida) 클론

## 소개

백준 온라인 저지 문제 번호에 solved.ac 티어 색을 칠해주는 크롬 확장 프로그램

## 미리 보기

![Preview](preview.png)

## 이용 방법

### 직접 설치하기

1. [TIERIMNIDA 클론](https://github.com/yehwankim23/tierimnida-clone/releases/download/v1.0.0/TIERIMNIDA_Clone_1.0.0.zip)을 다운로드하고 압축 해제한다.
2. 크롬 확장 프로그램 탭(`chrome://extensions`) 우측 상단에서 `개발자 모드`를 활성화시킨다.
3. `압축해제된 확장 프로그램을 로드합니다.`를 클릭하고 압축 해제한 `TIERIMNIDA 클론` 폴더를 선택한다.
